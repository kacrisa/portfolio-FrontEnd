Argentina Programa

2022 - Actualidad

Aplicación web full stack con el propósito de ser un portfolio web. Con arquitectura distribuida, diseño de interface (front end) que muestra la información, una base de datos que almacena los datos, APIs necesarias para proveer a través de internet la información (back end). Herramientas, lenguajes y protocolos utilizados: Angular, HTML, CSS, Boostrap, Spring Boot > Api Rest, Apache Tomcat, MySQL, Visul Studio Code, GIT, GitHub.
